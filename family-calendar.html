Alexa Integration

Add events hands-free with voice commands

Voice Command Examples

"Alexa, add Soccer Practice on Friday at 4pm for Jake"
Adds an event for a family member on a specific day/time
"Alexa, add Doctor Appointment on March 10th at 2pm"
Adds an event on a specific date
"Alexa, add Family Dinner tonight at 7pm"
Adds an event for today/tonight/tomorrow

Setup Instructions

1. Open the Alexa app → Skills & Games → search "IFTTT"
2. Enable IFTTT skill and connect your account
3. In IFTTT, create an Applet: If Alexa → Then Webhooks
4. Set trigger phrase: "add [EventName] on [Date] at [Time]"
5. Set webhook URL to the endpoint below
6. Save & test with the simulator below!
